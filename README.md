# 🌍 GeoScope Analytics: Global Country Intelligence System

## 📌 Overview
GeoScope Analytics is a data analysis project focused on exploring global country-level datasets.  
It extracts meaningful insights related to governance, geography, economy, and demographics.

---

## 📂 Dataset Description

The dataset contains structured information about countries across multiple dimensions.

### 🔑 Key Features (Columns)

- **Country** → Name of the country  
- **Capital** → Capital city  
- **Region / Subregion** → Geographic classification  
- **Population** → Total population  
- **Area** → Land area of the country  
- **Density** → Population density  
- **GDP / Economy Indicators** → Economic metrics  
- **Currency** → Official currency  
- **Languages** → Spoken languages  
- **Government Type** → Political system  
- **Head of State / Title** → Leadership information  
- **Independence / Historical Data** → Key historical attributes  

---

## 🎯 Objectives

- Analyze country-wise distributions  
- Compare economic and demographic indicators  
- Identify patterns across regions  
- Extract insights from governance structures  
- Build visual dashboards for decision-making  

---

## ⚙️ Feature Extraction

The project focuses on extracting and analyzing:

- Country-based metrics  
- Regional comparisons  
- Leadership roles and governance patterns  
- Economic indicators (GDP, density, etc.)  
- Demographic distributions
